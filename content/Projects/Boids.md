Checkout the project here 👉 https://project.ebelz.com/boids

An interactive simulation of flocking behavior using the Boids algorithm, built with React and TypeScript. Customize parameters like separation, alignment, and cohesion forces to see how they affect the flock's movement.
![[Screenshot 2026-01-26 at 6.26.45 PM.png]]
Based on the implementation by V. Hunter Adams outlined here: https://vanhunteradams.com/Pico/Animal_Movement/Boids-algorithm.html
